# PİKAREKS

Bu repo iki ayrı proje içerir:

- `frontend/` (Vite)
- `backend/` (Node.js + Express)

## Çalıştırma

### Backend
```bash
cd backend
npm i
npm run dev
```

`/api/health` kontrolü: `http://localhost:3001/api/health`

### Frontend
```bash
cd frontend
npm i
npm run dev -- --host 0.0.0.0
```
